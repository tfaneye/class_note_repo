Installing Git for Linux:
-------------------------

sudo apt-get update
sudo apt-get install git

Configuring Git
git config --global user.name "user_name" (Replace user_name with your GitLab user name)
git config --global user.email "email_address" (Replace email address with your GitLab email address)

<-------Ignore the lines below--------------->
Generating SSH Key:
ssh-keygen -t rsa -C "email_address"
cat ~/ .ssh/id_rsa.pub
xclip -sel clip < ~/ .ssh/id_rsa.pub (// this command copy the key to clipboard. May need to install xclip: sudo apt install xclip)
